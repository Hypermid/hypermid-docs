---
title: "Get Chains"
openapi: "GET /v1/chains"
description: "List all supported blockchains"
---

Returns the list of all blockchains supported by Hypermid, including EVM chains, Solana, Bitcoin, and Near Intents chains.

<ParamField query="type" type="string">
  Filter by chain type. Possible values: `EVM`, `SVM`, `UTXO`, `NEAR_INTENTS`.
</ParamField>
