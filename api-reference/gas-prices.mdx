---
title: "Get Gas Prices"
openapi: "GET /v1/gas-prices"
description: "Get current gas prices for supported chains"
---

Returns the current gas prices for the specified chain, useful for estimating transaction costs.

<ParamField query="chainId" type="number" required>
  The chain ID to get gas prices for (e.g., `1` for Ethereum).
</ParamField>
