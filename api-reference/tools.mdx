---
title: "Get Tools"
api: "GET /v1/tools"
description: "List available swap and bridge tools"
---

<CodeGroup>
```typescript TypeScript
const tools = await client.getTools();

const bridges = tools.data.tools.filter(t => t.type === "bridge");
```
