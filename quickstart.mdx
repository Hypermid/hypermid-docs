# Quickstart

This guide walks you through getting a quote and executing a cross-chain swap using the Hypermid API. By the end, you'll have a working integration.

## Prerequisites

- Node.js 18+ (for the TypeScript SDK)
- A wallet address on the source chain
- Tokens to swap

## Step 1: Install the SDK

<CodeGroup>
```bash npm
npm install @hypermid/sdk
```

```bash pnpm
pnpm add @hypermid/sdk
```

```bash yarn
yarn add @hypermid/sdk
```
</CodeGroup>

## Step 2: Initialize the Client

<CodeGroup>
```typescript TypeScript
import { Hypermid } from "@hypermid/sdk";

const client = new Hypermid({
  apiKey: "your-api-key", // Optional: omit for anonymous access (100 req/min)
});
```

```bash cURL
# No setup needed — just include the header in requests
# Anonymous: no header required
# Partner: add X-API-Key header
export HYPERMID_API_KEY="your-api-key"
```
</CodeGroup>

## Step 3: Get a Quote

Fetch the best available route for swapping ETH on Ethereum to USDC on Arbitrum.

<CodeGroup>
```typescript TypeScript
const quote = await client.getQuote({
  fromChain: 1,           // Ethereum
  toChain: 42161,         // Arbitrum
  fromToken: "0x0000000000000000000000000000000000000000", // ETH
  toToken: "0xaf88d065e77c8cC2239327C5EDb3A432268e5831",   // USDC on Arbitrum
  fromAmount: "100000000000000000", // 0.1 ETH in wei
  fromAddress: "0xYourWalletAddress",
});

console.log("Estimated output:", quote.data.estimate.toAmount);
console.log("Tool used:", quote.data.tool);
```

```bash cURL
curl -X GET "https://api.hypermid.io/v1/quote?\
fromChain=1&\
toChain=42161&\
fromToken=0x0000000000000000000000000000000000000000&\
toToken=0xaf88d065e77c8cC2239327C5EDb3A432268e5831&\
fromAmount=100000000000000000&\
fromAddress=0xYourWalletAddress" \
  -H "X-API-Key: your-api-key"
```

```go Go
package main

import (
    "fmt"
    "io"
    "net/http"
)

func main() {
    url := "https://api.hypermid.io/v1/quote?" +
        "fromChain=1&toChain=42161" +
        "&fromToken=0x0000000000000000000000000000000000000000" +
        "&toToken=0xaf88d065e77c8cC2239327C5EDb3A432268e5831" +
        "&fromAmount=100000000000000000" +
        "&fromAddress=0xYourWalletAddress"

    req, _ := http.NewRequest("GET", url, nil)
    req.Header.Set("X-API-Key", "your-api-key")

    resp, err := http.DefaultClient.Do(req)
    if err != nil {
        panic(err)
    }
    defer resp.Body.Close()

    body, _ := io.ReadAll(resp.Body)
    fmt.Println(string(body))
}
```
</CodeGroup>

## Step 4: Execute the Swap

<CodeGroup>
```typescript TypeScript
const execution = await client.execute({
  fromChain: 1,
  toChain: 42161,
  fromToken: "0x0000000000000000000000000000000000000000",
  toToken: "0xaf88d065e77c8cC2239327C5EDb3A432268e5831",
  fromAmount: "100000000000000000",
  fromAddress: "0xYourWalletAddress",
  toAddress: "0xYourWalletAddress",
});

// Sign and send the transactionRequest via your wallet
if (execution.data.transactionRequest) {
  const tx = await wallet.sendTransaction(execution.data.transactionRequest);
  console.log("Transaction hash:", tx.hash);
}

// For deposit routes: send to the provided address
if (execution.data.depositAddress) {
  console.log("Send tokens to:", execution.data.depositAddress);
  console.log("Memo:", execution.data.memo);
}
```

```bash cURL
curl -X POST "https://api.hypermid.io/v1/execute" \
  -H "Content-Type: application/json" \
  -H "X-API-Key: your-api-key" \
  -d '{
    "fromChain": 1,
    "toChain": 42161,
    "fromToken": "0x0000000000000000000000000000000000000000",
    "toToken": "0xaf88d065e77c8cC2239327C5EDb3A432268e5831",
    "fromAmount": "100000000000000000",
    "fromAddress": "0xYourWalletAddress",
    "toAddress": "0xYourWalletAddress"
  }'
```
</CodeGroup>

## Step 5: Track Status

<CodeGroup>
```typescript TypeScript
const status = await client.getStatus({
  txHash: "0xYourTransactionHash",
  fromChain: 1,
  toChain: 42161,
});

console.log("Status:", status.data.status);
// Possible values: "PENDING", "DONE", "FAILED", "NOT_FOUND"
```

```bash cURL
curl -X GET "https://api.hypermid.io/v1/status?\
txHash=0xYourTransactionHash&\
fromChain=1&\
toChain=42161" \
  -H "X-API-Key: your-api-key"
```
</CodeGroup>

<Tip>
  For deposit-based routes, use `GET /v1/execute/deposit/status` with the `depositId` returned from the execute step instead of `GET /v1/status`.
</Tip>
