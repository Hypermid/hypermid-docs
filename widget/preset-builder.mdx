Use the interactive builder below to browse all supported chains and tokens. Select your desired from/to chain and tokens, then copy the generated embed code.

<iframe
  src="https://www.hypermid.io/widget/preset-builder"
  width="100%"
  height="850"
  frameborder="0"
  allow="clipboard-write"
  style="border: none; border-radius: 8px; margin-top: 16px; min-height: 800px;"
></iframe>

<Info>
  The builder fetches chains and tokens directly from the LiFi API at runtime, so you always see the complete and up-to-date list. Hypermid supports **90+ chains** and **20,000+ tokens**.
</Info>

## How It Works

1. **Search chains** — Type a chain name, symbol, or ID in the search box
2. **Select From Chain** — Pick the source chain, then select a token from the list below it
3. **Select To Chain** — Pick the destination chain and token
4. **Copy code** — Use the generated **Script Tag** or **Iframe** embed code on the right

## Using Custom Chain & Token IDs

If you know the chain ID and token contract address, you can pass them directly without using this builder:

### Iframe URL Parameters

```
https://www.hypermid.io/widget?embed=1
  &fromChain=CHAIN_ID
  &toChain=CHAIN_ID
  &fromToken=TOKEN_ADDRESS
  &toToken=TOKEN_ADDRESS
```

### Script Tag Config

```javascript
HypermidWidget.init({
  containerId: "hypermid-widget",
  defaultFromChain: 1,          // Chain ID (number)
  defaultToChain: 42161,        // Chain ID (number)
  defaultFromToken: "0x...",    // Token contract address
  defaultToToken: "0x...",      // Token contract address
});
```

### Token Address Notes

- **Native tokens** (ETH, BNB, MATIC, etc.) use `0x0000000000000000000000000000000000000000`
- **ERC-20 tokens** use the token's contract address on that chain
- **Solana tokens** use the SPL token mint address
- Token addresses are **chain-specific** — USDC on Ethereum has a different address than USDC on Arbitrum

## API Endpoints

You can also fetch chain and token data programmatically:

```bash
# Get all supported chains
curl https://api.hypermid.io/v1/chains

# Get tokens for specific chains
curl "https://api.hypermid.io/v1/tokens?chains=1,42161,8453"

# Get a specific token
curl "https://api.hypermid.io/v1/tokens?chains=1&search=USDC"
```

See the [API Reference](/api-reference/chains) for full documentation.
